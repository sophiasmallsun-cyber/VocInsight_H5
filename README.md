# VocInsight BI — H5 移动端适配版本

用户声量分析后台 (User Voice Volume Analysis Dashboard)，已适配移动端 H5 响应式布局。

View your app in AI Studio: https://ai.studio/apps/d291a750-6429-461c-88e9-f63df4a756c2

## H5 改造内容

- 📱 侧边栏 → 汉堡菜单 + 滑出式抽屉面板
- 📊 底部 Tab 导航栏
- 🎯 响应式筛选器布局
- 📋 数据表格横向滚动
- 📈 图表自适应尺寸
- 🔤 中英文双语支持

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
